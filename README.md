# Project-06-Hybrid-Identity-using-Microsoft-Entra-Connect
Implemented a hybrid identity environment by synchronizing on-premises Active Directory with Microsoft Entra ID.

# Tasks Performed:

1. Installed and configured Windows Server Domain Controller.
2. Created Organizational Units (OUs) and test users.
3. Installed Microsoft Entra Connect Sync.
4. Connected on-premises Active Directory to Microsoft Entra ID.
5. Configured Password Hash Synchronization.
6. Verified user synchronization to Microsoft Entra ID.
7. Configured the Hybrid Identity Wizard.
8. Tested user sign-in using synchronized credentials.
9. Verified synchronization using Synchronization Service Manager and Microsoft Entra Admin Center.
10. Troubleshot authentication issues during Entra Connect configuration (AADSTS700016) and successfully completed the deployment.

1. Windows Server

3. DNS
