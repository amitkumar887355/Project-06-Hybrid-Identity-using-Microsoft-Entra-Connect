# Project-06-Hybrid-Identity-using-Microsoft-Entra-Connect
Implemented a hybrid identity environment by synchronizing on-premises Active Directory with Microsoft Entra ID.

# Tasks Performed:

Installed and configured Windows Server Domain Controller.
Created Organizational Units (OUs) and test users.
Installed Microsoft Entra Connect Sync.
Connected on-premises Active Directory to Microsoft Entra ID.
Configured Password Hash Synchronization.
Verified user synchronization to Microsoft Entra ID.
Configured the Hybrid Identity Wizard.
Tested user sign-in using synchronized credentials.
Verified synchronization using Synchronization Service Manager and Microsoft Entra Admin Center.
Troubleshot authentication issues during Entra Connect configuration (AADSTS700016) and successfully completed the deployment.
